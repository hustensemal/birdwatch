# BirdWatch – Chat-Commands

Statische Command-Übersicht für den privaten Vogelstream auf Twitch und YouTube.

## Veröffentlichung mit GitHub Pages

1. Den **Inhalt dieses Ordners** in das Repository `hustensemal/birdwatch` hochladen – nicht den umschließenden Ordner selbst.
2. Im Repository **Settings → Pages** öffnen.
3. Unter **Build and deployment** als Quelle **Deploy from a branch** wählen.
4. Branch **main** und Ordner **/(root)** auswählen und speichern.
5. Nach der Veröffentlichung ist die Seite unter dieser Adresse erreichbar:
   `https://hustensemal.github.io/birdwatch/`

## Commands ändern

Die Einträge stehen in `index.html`. Die Suche und Kategorie-Filter werden durch `script.js` gesteuert.

## Datenschutz

Die Seite setzt keine Cookies, verwendet kein Tracking und lädt keine externen Inhalte. Suchbegriffe werden nur lokal im Browser verarbeitet.

## Indexierung

Die HTML-Datei enthält `noindex`-Anweisungen. Das erschwert die Aufnahme in seriöse Suchmaschinen, ist aber keine Zugangssperre. Jeder, der den Link kennt, kann die Seite aufrufen.

## Bedienung

- Kategorien lassen sich auf- und zuklappen.
- Eine Suche öffnet automatisch nur die Bereiche mit passenden Treffern.
- Ein Klick auf einen Command kopiert ihn in die Zwischenablage.
